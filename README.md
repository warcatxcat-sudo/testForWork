Весь код работает в тестовом режиме, в еальном проекте input() и print() будут заменены на взаимодействие с другими API, например print() будет заменён на запрос в frontend интерфейс с передачей данных из БД.

Схема управления:
У каждого пользователя в БД есть поле в котором содержаться права доступа(не путать с полем доступных ресурсов), в нём описаны права пользователя которые будут проверяться при соответствующих запросах, редактиование других пользователей будет разрешено только пользователям "рангом" выше редактируемого. Удаление и возвращение аккаунтов из soft-delete так же возможно пользователем с нужными правами.
